# NDBI Change Detection (2015–2025) – Mumbai

## 📌 Project Overview
This project analyzes urban expansion in Mumbai using NDBI (Normalized Difference Built-up Index) for the years 2015 and 2025.

## 🛠 Tools Used
- QGIS (PyQGIS)
- Python (NumPy, Matplotlib)
- Landsat Data

## 📊 Workflow
1. Data preprocessing and clipping using AOI
2. NDBI calculation
3. Classification into:
   - Vegetation
   - Transition Land
   - Built-up
4. Area and percentage calculation
5. Visualization using maps and graphs

## 📈 Results
- Built-up area increased significantly
- Transition land decreased
- Very low vegetation cover observed
